# PopX - Mobile App Interface

A pixel-perfect React implementation of the PopX mobile app interface design from Adobe XD.

## Screens

1. **Landing Screen** - Welcome page with scattered dots and navigation buttons
2. **Login Screen** - Email and password login form
3. **Signup Screen** - Multi-field registration form with agency radio selection
4. **Profile Screen** - Account settings with avatar and user details

## Tech Stack

- **React** (v19) with Vite
- **React Router DOM** for client-side navigation
- **Vanilla CSS** with design tokens
- **Google Fonts** (Rubik)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
